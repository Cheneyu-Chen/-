# 快速启动指南

## 首次使用 - 三种方法选择

### 方法 1：最简单（推荐）
直接双击 `一键启动.bat` 文件，脚本会自动：
- ✓ 检查 Python 环境
- ✓ 创建虚拟环境
- ✓ 安装所有依赖库
- ✓ 启动应用

**适用于：** 大多数 Windows 电脑

---

### 方法 2：如果方法1报错
使用 PowerShell 版本，双击 `一键启动.ps1` 文件

**注意：** 
- 如遇权限问题，请右键 -> 用 PowerShell 以管理员身份运行
- 首次运行 PowerShell 脚本可能需要更改执行策略

**适用于：** 特殊网络环境或代理环境

---

### 方法 3：命令行手动启动
如果上述两种方法都不行，请打开命令窗口并执行：

```bash
# 创建虚拟环境
python -m venv .venv_local

# 激活虚拟环境
.venv_local\Scripts\activate

# 安装依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 启动应用
python -m app.main
```

---

## 环境要求

### 必需：
- **Python 3.10 或更高版本**

### 如何安装 Python：
1. 访问 [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. 选择 Python 3.10+ 版本下载
3. **重要：安装时必须勾选 "Add Python to PATH" 选项**
4. 完成安装后重启电脑

### 验证安装：
打开命令窗口，输入以下命令，应该看到版本号：
```bash
python --version
```

## 常见问题

### Q1: 双击脚本没有反应
**A:** 请确保：
1. Python 已正确安装并添加到 PATH
2. 有网络连接
3. 文件夹路径中没有特殊字符

重试：先删除 `.venv_local` 文件夹，再运行脚本

### Q2: 提示"未找到 Python"
**A:** 说明 Python 未添加到 PATH，请：
1. 卸载 Python
2. 重新安装，**必须勾选** "Add Python to PATH"
3. 安装完成后重启电脑

### Q3: 依赖安装超时/失败
**A:** 可能是网络问题，尝试：
1. 检查网络连接
2. 如果使用代理/VPN，请临时关闭后重试
3. 删除 `.venv_local` 文件夹，再运行脚本

### Q4: 应用启动后立即关闭
**A:** 请检查：
1. 命令窗口是否有错误提示
2. GPU/显卡驱动是否为最新版本
3. 运行方法3（手动命令行）来查看具体错误

### Q5: 我在 Mac 或 Linux 上
**A:** 请参考 `README.md` 中的 Linux/Mac 启动说明

## 环境文件说明

- **`一键启动.bat`** — Windows 批处理脚本（推荐首选）
- **`一键启动.ps1`** — PowerShell 脚本（备选方案）
- **`.venv_local/`** — 虚拟环境目录（首次运行自动创建）
- **`requirements.txt`** — Python 依赖库列表

## 故障排查步骤

如果以上都不行，请按顺序尝试：

1. **完全清除环境**
   ```bash
   # 删除这个文件夹
   .venv_local
   ```

2. **用管理员身份启动**
   - Windows: 右键 `一键启动.bat` -> 以管理员身份运行

3. **手动安装依赖**
   ```bash
   python -m pip install --upgrade pip
   pip install PySide6 numpy scipy matplotlib
   ```

4. **提交错误信息**
   - 运行方法3（命令行版本），截图保存错误提示
   - 包含 Python 版本 (python --version)
   - 包含系统版本 (Windows 几)
   - 联系技术支持
