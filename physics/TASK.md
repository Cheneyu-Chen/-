请开发一个用于全国大学生物理实验竞赛（创新）自选题2“教学资源和虚仿”的 Python 桌面软件，主题为：

# 声场与振动可视化虚拟仿真平台

## 总目标
做一个尽可能完整、美观、中文友好、交互良好的桌面应用，适合比赛展示、录屏、后续继续开发。

## 硬性要求
1. 使用 Python 开发。
2. 中文界面，尽量避免乱码风险。
3. 图形界面要美观、结构清晰、适合 1920x1080 展示。
4. 要能在当前项目目录下运行。
5. 项目文件全部放在当前“项目文件”目录内。
6. 尽量不要依赖过于冷门的库。优先使用：
   - PySide6
   - numpy
   - scipy
   - matplotlib
7. 给出 requirements.txt 和 README.md。
8. 给出一个一键启动方式，例如 main.py。
9. 代码结构清晰，便于后续人工修改。
10. 完成后请不要只写骨架，至少做出可运行的首版完整软件。

## 功能要求
### 1. 主界面
- 中文标题
- 简介卡片
- 模块导航
- 深色或现代风格 UI

### 2. 一维驻波仿真模块
至少支持：
- 调节频率
- 调节阻尼
- 调节振幅
- 调节激励位置或模式参数
- 播放/暂停/重置
- 动态显示波形
- 显示节点/腹点或振幅分布
- 给出中文解释

### 3. 二维振动模态模块
至少支持：
- 展示若干典型二维模态
- 显示热图
- 显示节点线
- 切换模态阶数
- 切换边界或模式类型（如矩形板/圆膜，哪怕其中一个是预留扩展也可以）
- 给出中文解释

### 4. 共振扫描模块
至少支持：
- 设置扫描频率范围
- 生成响应曲线
- 标出共振峰
- 中文说明

### 5. 教学案例模块
至少内置几个案例：
- 基本驻波形成
- 模态变化
- 阻尼影响
- 共振峰变化

### 6. 软件体验要求
- 参数控件联动流畅
- 图表刷新清晰
- 尽量不要卡顿
- 中文字体显示正常
- 代码里注意适配中文 Windows 路径
