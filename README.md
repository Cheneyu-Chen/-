# 声场与振动可视化虚拟仿真平台

本项目是面向大学物理教学、声学实验和科普展示的可视化虚拟仿真平台。平台围绕机械波、声场、驻波、共振、克拉尼图形和声学超材料等内容，提供可调参数、动态演示、三维可视化、实验图片分析和自动报告生成能力。

## 项目亮点

- **动态仿真**：支持一维驻波、二维模态、三维声波、声场干涉和共振扫描的动态演示。
- **三维声波实验**：展示点声源球面波、双声源三维干涉和矩形房间驻波模态。
- **进阶声学实验**：包含单缝声衍射、声子晶体带隙、亥姆霍兹共鸣吸声等内容。
- **完整展示界面**：提供启动页、玻璃拟态 UI、作品总览页、教学案例页和实验对比页。
- **增强分析工具**：支持实验照片导入、相似度计算、自定义多边形边界、有限差分薄板求解、二维声学超材料阵列和自动生成实验报告。
- **中文适配**：界面、报告和实验案例均按中文展示与命名。

## 快速启动

Windows 下推荐直接双击：

```bat
一键启动.bat
```

备用入口：

```bat
启动软件.bat
```

精简入口：

```bat
快速启动.bat
```

脚本会自动检查本地虚拟环境和依赖。如果依赖缺失，会根据 `requirements.txt` 自动安装。

## 主要功能模块

| 模块 | 功能 |
|---|---|
| 平台总览 | 展示作品定位、教学痛点、解决方案和成果输出 |
| 作品总览 | 汇总实验矩阵、核心特点和演示路线 |
| 平台设计 | 说明系统流程、模型层次、交互设计和平台特点 |
| 一维驻波 | 演示不同边界条件下的驻波、节点和波腹 |
| 二维模态 | 展示矩形、圆形、三角形膜/薄板模态和克拉尼图形 |
| 共振扫描 | 计算受迫振动幅频响应、共振峰和阻尼影响 |
| 进阶声学 | 包含干涉、衍射、声子晶体带隙和亥姆霍兹吸声 |
| 三维声波 | 动态展示三维球面波、双声源干涉和房间模态 |
| 教学案例 | 提供 12 个一键跳转预设案例 |
| 实验对比 | 支持实验验证流程、误差分析和课程联系说明 |
| 增强工具 | 支持照片相似度、多边形有限差分、二维超材料和报告生成 |

## 项目结构

```text
.
├─ app/                    # 主程序源码
│  ├─ core/                 # 数值模型与算法
│  ├─ pages/                # PySide6 页面
│  └─ widgets/              # 通用组件、画布和动态玻璃背景
├─ outputs/
│  └─ 实验案例/             # 整理好的实验图片、分析图和报告
├─ tools/                   # 实验图片下载、分析和整理脚本
├─ requirements.txt         # Python 依赖
├─ 启动软件.bat             # 推荐启动入口
├─ 一键启动.bat             # 备用启动入口
├─ 快速启动.bat             # 精简启动入口
└─ 项目说明.md              # 简要目录说明
```

## 实验案例

整理后的实验案例位于：

```text
outputs/实验案例
```

当前包含 12 个案例文件夹，例如：

- `01_Slinky弹簧驻波`
- `02_弦驻波实验`
- `03_Kundt管声学驻波`
- `05_矩形克拉尼板`
- `07_圆形克拉尼板三环节点`
- `10_共振曲线`
- `12_二维超材料阵列声场`

每个案例文件夹包含：

- `实验照片.jpg/png` 或 `生成参考图.png`
- `分析结果图.png`
- `本实验说明.md`

总览文件：

- `outputs/实验案例/总览报告.md`
- `outputs/实验案例/总相似度结果.csv`

## 依赖安装

依赖写在 `requirements.txt`：

```text
PySide6
numpy
scipy
matplotlib
```

通常不需要手动安装，启动脚本会自动处理。若需要手动安装：

```powershell
..\.venv_local\Scripts\python.exe -m pip install -r requirements.txt
```

## 说明与注意

- `outputs/实验案例` 中部分图片来自公开实验图片来源，部分为平台生成参考图；生成图均已在说明文件中明确标注。
- 实验照片相似度用于辅助教学分析和报告讨论，不应作为唯一物理评价标准。
- `.venv_local` 和 `.pip_temp` 为本地运行环境目录，不属于平台核心源码。
